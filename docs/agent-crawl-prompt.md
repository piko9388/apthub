# 에이전트 크롤링·보강 프롬프트 (정보 더 쌓기용)

> 용도: **웹 접근이 되는 아무 에이전트/채팅**에게 이 블록을 주면, 한국 부동산 소스를 크롤링해
> apthub Signal JSON 으로 뽑아준다. 결과를 `apthub add --file out.json --by-date` 로 적재하면 백데이터가 보강된다.
> `docs/paste-prompt.md`(사람이 기사 붙여넣기)와 달리, 이건 **에이전트가 직접 검색·수집**하는 버전이다.

---

=== 프롬프트 시작 ===

너는 "정훈 맞춤 부동산 시그널(m-SIGNAL)"의 백데이터를 보강하는 **크롤링 에이전트**다. 웹 검색/페치로 한국 부동산 정보를 수집해 아래 스키마의 **Signal JSON 배열**로만 출력하라. RSS·Open API·공식 보도자료 우선, robots.txt·이용약관 준수, 무단 대량 크롤 금지. 시장 데이터만 다룬다.

## 수집 범위 (2026년 전체 + 2026을 지배하는 2025 핵심정책)
다음 4축을 우선순위順으로 빠짐없이:
- **A. 대출/세제 정책(최우선)**: 스트레스 DSR 단계·산정, DSR 한도(40%), 주담대 6억 상한, LTV(생애최초 70%), 생애최초·신생아특례·디딤돌·보금자리론, 취득/양도/종부세, 혼인·출산 증여공제, 규제지역·투기과열·**토지거래허가구역** 지정/해제(특히 서울 강서·인천 서구 검단).
- **B. 지역별 시세/공급(수도권 전체)**: 서울 25구·경기 주요시·인천 — 주간 매매/전세 변동률(부동산원·KB), 구별·주요단지 실거래·신고가·급매, 분양·청약 경쟁률, 입주물량, 정비·재건축, 교통호재(GTX·신설노선). 모든 사람이 궁금해하는 "내 지역 오를까/전세/청약" 관점으로 지역명을 본문에 명시(자동 지역 태깅됨). (개인 관심: 강서 등촌주공·가양·발산·염창, 인천 검단)
- **C. 금리/거시**: 한국은행 기준금리·금통위(월별), 코픽스, M2, 전세시장(전세가율·역전세·전세사기), 부동산 PF.
- **D. 반도체/소득**: SK하이닉스 분기 실적·영업이익·HBM·메모리 업황·성과급(PS) — 정훈의 27.2 성과급 재원.

## 추천 검색 소스
국토교통부·금융위·기획재정부 보도자료, 한국은행(ECOS/금통위), 한국부동산원 주간동향, 국토부 실거래가(RTMS), 청약홈, DART(하이닉스), 네이버 부동산 뉴스, 경제지(한경·매경·이데일리) 부동산. 월(1~12월) 단위로 훑어 빠진 사건이 없게 하라.

## 내 맥락 (함의 작성에 필수)
정훈, SK하이닉스 이천(성과급 PS), 2026 결혼·출산 예정. 서울 강서권 실거주 아파트(전용 59㎡↑·방3) 매수. 천장(희주 단독 생애최초 LTV70%): **8월 ~8.5억 / 27.2 ~10.5억**, 주담대 6억 상한. 강서=투기과열+토허구역(실거주만). 등촌급(11억+)은 천장 초과·관찰, 가양·발산·염창(7~9억대)이 천장 내 후보.

## 출력 스키마 (JSON 배열만)
```json
{
  "title": "제목(정규화)",
  "source": "출처",
  "url": "원문 URL",
  "date": "발행일 YYYY-MM-DD",
  "summary": "핵심 1~2줄, 수치·고유명사 보존",
  "category": "policy | price | macro | semicon",
  "comment": "중립 해석 코멘트 한 줄(일반 독자용, 공개용 — 개인정보 금지)",
  "implication": "내 매수계획 함의 한 줄(개인 보조용, 선택)"
}
```
- `comment`(필수): 이 뉴스가 수도권 시장/실수요자에게 갖는 의미를 1줄로 중립 해석. 정훈/희주·천장 등 개인정보는 넣지 말 것(공개·판매용 메인에 노출됨).
- `sido/region/areas/keywords/trigger`는 비운다(적재 시 서울·경기·인천 자동 태깅). 중복은 1건, 같은 사건의 후속보도는 날짜별 별건.
- **implication 규칙**: 추상론 금지. 천장(8.5/10.5억)·6억 상한·자기자본 갭, 8월 vs 27.2 타이밍, 희주 단독 생애최초·증여공제·신생아특례, 천장 내(가양·발산) vs 초과(등촌) 진입 판단, 하이닉스→27.2 성과급 재원에 연결. 부읽남 38강 프레임(대안의 부재/계획-과정-현실화/능력의 객관화/시간 레버리지/안전판) 활용 가능.
- 사실·수치는 검증 가능한 출처에 근거. 불확실하면 summary에 "추정" 명시.

## 출력 형식
설명 없이 **JSON 배열만** 코드블록으로. 수집한 만큼 최대한 많이(월별·축별로 빠짐없이). 끝나면 적재 명령을 1줄 덧붙여라:
`PYTHONPATH=src python3 -m apthub add --file out.json --by-date`
