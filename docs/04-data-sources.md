# 04 — 데이터 소스

> RSS·Open API 우선. robots.txt·이용약관 준수, 무단 대량 크롤 금지. (실제 수집 가능 여부는 새 세션에서 조사·검증할 것)

## 정책 / 제도
| 소스 | 대상 | 방식(예상) |
|---|---|---|
| 국토교통부 보도자료 | 부동산 대책·규제지역·공급 | 사이트/RSS |
| 금융위원회 | DSR·LTV·대출규제 | 사이트/보도자료 |
| 기획재정부 | 세제(취득·양도·종부·증여) | 보도자료 |
| 한국은행 | 기준금리·통화량 | 사이트/Open API(ECOS) |
| 법제처 입법예고 | 시행령·규칙 개정 예고 | 사이트 |

## 시세 / 실거래 / 청약
| 소스 | 대상 | 방식 |
|---|---|---|
| 국토부 실거래가 공개시스템(RTMS) | 아파트 실거래 | **Open API**(공공데이터포털) |
| 한국부동산원 R-ONE | 주간/월간 시세·통계 | **API** |
| KB부동산 | KB시세·주간동향 | 사이트 |
| 청약홈(APT2you) | 분양·청약 일정·경쟁률 | 사이트 |

## 뉴스 / 커뮤니티
| 소스 | 대상 | 방식 |
|---|---|---|
| 네이버 뉴스(부동산 섹션) | 종합 | RSS/검색 |
| 경제지(한경·매경·이데일리 부동산) | 정책·시장 | RSS |
| 부읽남 등 부동산 유튜브/블로그 | 해설·관점 | RSS |

## 반도체 / 소득 연동
| 소스 | 대상 | 방식 |
|---|---|---|
| DART(전자공시) | SK하이닉스 실적·공시 | **Open API** |
| 반도체 전문 뉴스 | HBM·메모리 업황 | RSS/검색 |

## 수집 원칙
- 공공데이터포털·ECOS·DART는 **API 키 발급** 후 사용(가장 안정적·합법).
- 뉴스는 RSS 우선, 없으면 검색 API. 본문 전체보다 제목·요약·링크 위주 수집.
- 출처 URL·발행일시 보존, 중복 제거(URL/제목 해시).
