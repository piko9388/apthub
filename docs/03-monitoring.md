# 03 — 모니터링 대상·트리거·키워드

## A. 대출 / 세제 정책 (최우선)
- 스트레스 DSR 단계·산정방식, DSR 한도(현 40%)
- 주담대 한도(현 수도권 **6억 상한**, 6.27 대책), LTV(규제지역 무주택 40% / **생애최초 70%**)
- 생애최초·신생아 특례·디딤돌·보금자리론 요건·금리
- 취득세·양도세(1주택 비과세 12억)·종부세, **혼인·출산 증여공제**
- 규제지역(투기과열·조정대상) 지정/해제 — 특히 **서울 강서구, 인천 서구(검단)**

**키워드**: `DSR` `스트레스DSR` `LTV` `주담대 6억` `생애최초` `신생아 특례` `디딤돌` `보금자리론` `규제지역` `조정대상지역` `취득세` `양도세` `종부세` `증여공제`

## B. 시세 / 공급 (강서 중심)
- 실거래가(국토부 RTMS), KB·부동산원 주간시세
- 입주물량, 분양·청약, 재건축·정비사업 진행(등촌주공 정비, 가양 CJ부지, 9호선 급행, GTX)

**키워드**: `강서 실거래` `등촌주공 시세` `강서 분양` `강서 입주물량` `재건축 안전진단` `정비사업` `신고가` `급매`

## C. 거시 / 금리
- 한국은행 기준금리, 주담대 금리(코픽스), 통화량(M2)

**키워드**: `기준금리` `한국은행` `코픽스` `주담대 금리` `역전세` `전세사기` `M2 통화량`

**키워드**: `SK하이닉스 실적` `HBM` `D램 가격` `메모리 업황` `하이닉스 성과급`
