# APT-SIGNAL 벤치마크 리뷰 — 탑티어 리서치 대비 개선안

> 국내·해외 최상위 부동산 데이터/리서치 시스템과 비교해 현재 APT-SIGNAL의 약점과 개선 우선순위를 정리.
> (웹리서치 기반, 2026-06 시점. 출처 URL은 각 항목 말미.)

## 1. 탑티어는 데이터를 어떻게 보여주는가

### 국내
- **한국부동산원 R-ONE** (국가승인통계, 정책 벤치마크): 변동률 시계열 라인차트 + 통계지도(choropleth). **규모별** 분리(전용 40/60/85/130㎡), 월세를 준월세·준전세까지 분해. 기준 **2024.1=100**(표본확대 시 리베이스), 별도 **공동주택실거래가격지수**(2021=100, 실거래 기반). 주간(목)·월간.
- **KB 데이터허브** (시장·소비자 벤치마크): **통계기상도**(지도에 날씨 메타포 = 상승/하락), 시계열 Excel. **5분위(가격대별)** 평균가 + **5분위 배율**(상/하위 격차). 심리지수 — **매수우위지수·매매전망지수(0~200, 중앙 100)**. KB선도50지수(2022.1=100). 기준 2022.1.10=100. 주간(금)·월간.
- **아실(asil)**: **여러 단지 가격 비교 그래프**(2006~현재 매매·전세 오버레이), 입주물량·매물증감·갭투자·외지인 매입. 구별/평형대별/가격대별 세분이 가장 깊음.
- **부동산플래닛**: **평당가 컬러맵 + 노후도(건물연식) 지도**, AI추정가(정확도 ~70%/오차 20% 공개). 전 유형(토지·상업 포함).
- **호갱노노**: 실거래 지도 + **평형별 시세**, 경사·일조량 3D·학원가 히트맵, 시간단위 실시간.

### 해외
- **S&P Case-Shiller**: **반복매매(repeat-sales)** 지수, **2000.1=100 인덱스 라인**, **저/중/고가 3분위 tier 지수**, NBER 경기침체 음영. 월간(3개월 이동).
- **FRED**: 모든 시계열을 인덱스(임의 기준일 100 재설정)·YoY%·실질(CPI 디플레이트) 변환, 경기침체 음영 자동, CSV/Excel/API.
- **RICS**: 설문 **순지수(net balance) = 상승응답% − 하락응답%, −100~+100**(폭이 아닌 '확산도'). 신규수요·신규매물·가격·3·12개월 전망을 각각 시계열.
- **Knight Frank / Savills**: **prime = 시장 상위 5%(자산가치 기준)** 정의로 prime/mainstream을 분리, 인덱스+도시 랭킹 바. Savills는 자본가치·임대료·수익률 3종.

**공통 관통**: ① 숫자 나열이 아니라 **인덱스(기준 100) 라인·sparkline**, ② **세그먼트 분리**(가격대/면적대/tier/prime), ③ 방법론·기준시점 **명시**, ④ 이벤트·국면 **주석(음영/라벨)**.

## 2. APT-SIGNAL 약점 진단

| 영역 | 현재 | 약점 | 비교 대상 |
|---|---|---|---|
| 가시화 | 지표를 숫자 타일로 표시 → **sparkline 추가(이번)** | 인덱스(기준100) 라인·YoY·tier 비교 부재 | Case-Shiller/FRED/asil |
| 세그먼테이션 | sido·구 단위만 | **가격대별·면적대별·평당가·세대수 없음** | R-ONE 규모별, KB 5분위 |
| 출처 엄밀성 | 다수 지표가 '언론'(부동산원 인용) | **1차(R-ONE/KB/ECOS/data.go.kr API) 직접 매칭 필요** | 전 기관 |
| 시계열 깊이 | 월 1~수 포인트 | 12~24개월 연속 시계열 얇음 | 전 기관 |
| 방법론 투명성 | 기준·정의 표기 약함 | 기준시점·산정식(예: 매수우위 0~200) 노출 필요 | KB/Case-Shiller |
| 정합(뉴스↔지표) | 과열어/약세어 빈도(이번) | 지수 라인 위 **이벤트 점(관련 기사) 주석**으로 진화 | FRED add-line, RICS |
| 심리 | 없음 | 매수우위·전망지수(0~200) 수집·게이지 | KB |

## 3. 개선 우선순위

### P0 (즉시·핵심)
- **[진행] sparkline 추이선** — 모니터링 타일에 인라인 SVG 라인(완료). 다음: **기준 100 인덱스 라인**으로 정규화(변동률 누적 → 지수).
- **가격대/면적대 세그먼트** — 대출·세제 정책 경계로 가격대 밴드(6/9/15/25억) + 면적대(40/60/85/130㎡, R-ONE 표준) 분리 관리. 평당가·세대수 필드 추가.
- **1차 출처 매칭** — 지표는 언론 인용 대신 R-ONE/KB/ECOS/data.go.kr API 원본과 매칭(confidence "공식"). 기사·이벤트는 그 지표 라인 위에 점으로 연결.

### P1
- **이벤트 주석 플롯** — 지수 sparkline 위에 정책·신고가 이벤트를 점으로 찍고 관련 기사 링크(FRED add-line / Case-Shiller 음영 스타일).
- **심리지수** — KB 매수우위·매매전망(0~200, 중앙 100) 수집 + 게이지 시각화.
- **전세가율·PIR** — 부담·갭 분석(KB/부동산원/HUG houstat).
- **방법론 푸터** — 각 지표에 기준시점·산정식·출처 cadence 명시.

### P2
- 분양가(HUG)·경매 낙찰가율·청약경쟁률·미분양 추세를 공급·심리 보조지표로.
- 단지 비교(아실식 다단지 오버레이), 평당가 지도(부동산플래닛식).
- 인덱스 YoY/실질(CPI) 토글.

## 4. 트리거(🔴🟡) 재정의 — RICS 'net balance'에서 차용
현재 룰이 개인(강서·검단·하이닉스) 중심 → **공개 기준으로 재정의**(사용설명서 반영):
- **🔴 즉시**: 대출·세제·규제지역 등 **제도 변경**, **기준금리 변경**, 주요 단지 **신고가** 등 시장 전체 즉시 영향.
- **🟡 주목**: 코픽스·주담대 금리 변동, 공급·입주·분양, 정비사업 진전 등 **추세 점검**.
- (후속) RICS식 **순지수**(상승기사% − 하락기사%, −100~+100)를 주차별 흐름 지표로 도입 검토.
