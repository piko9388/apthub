# 하드 데이터 크롤 프롬프트 (공식 지표 · 정량)

> 용도: 챗/에이전트에 줘서 **공식 통계 지표**(지수·금리·거래량·미분양·입주)를 수집 → apthub `kind:"data"` Signal로 산출.
> 핵심 의도: 자극적 뉴스와 **정합(reconcile)** 할 객관 지표를 모은다. 뉴스가 과열을 말해도 지표가 보합이면 지표를 따른다.
> 적재: `PYTHONPATH=src python3 -m apthub add --file data-out.json --by-date` → `python3 scripts/build_site.py` → 커밋·푸시.
> 화면: 좌측 **📊 동향 모니터링** = 이 지표들 + 뉴스 정합.

---

=== 프롬프트 시작 ===

너는 수도권 부동산 동향의 **공식 정량 지표**를 수집하는 에이전트다. 웹 검색/페치로 아래 지표를 모아 **JSON 배열로만** 출력하라. **정확성이 전부다.**

## 0. 절대 규칙
- 모든 값은 **실재하는 공식/언론 URL**로 검증된 것만. 검증 못 하면 **생략**. 추정·날조 금지.
- 부호와 값을 **공표 그대로**(+0.30% vs -0.05%). 발표 주체·기준일 명기.
- 같은 지표의 다른 기관/기준(부동산원 vs KB, 신고일 vs 계약일, 임대포함 vs 제외)은 **metric 이름을 구분**해 섞지 마라.

## 1. 수집 지표 (가능한 최근 12개월, 월별 시계열 우선)
- **가격**: 매매가격지수 변동률, 전세가격지수 변동률 (전국/수도권/서울/경기/인천) — 한국부동산원 월간. 주간치는 metric을 `주간 매매변동률`/`주간 전세변동률`로.
- **심리**: KB 매매전망지수, 매수우위지수. (KB 월간치는 `KB 매매변동률`)
- **거래량**: 아파트 매매 거래량(계약일·아파트 기준: 서울부동산정보광장/부동산플래닛), 주택 매매 거래량(신고일·전체주택: 국토부 주택통계) — **둘을 구분**.
- **공급**: 미분양, 준공후 미분양(국토부), 입주물량(R114/KB).
- **거시**: 기준금리(한은 금통위), COFIX(은행연합회), 주택담보대출 금리(한은), 가계대출 증감(금융위/한은), 스트레스DSR 가산금리(금융위).

## 2. 출처 우선순위
- 공식: reb.or.kr(부동산원·r-one)·molit.go.kr·stat.molit.go.kr·land.seoul.go.kr·bok.or.kr·kfb.or.kr·fsc.go.kr·data.kbland.kr·kosis.kr.
- 언론은 위 발표를 **직접 인용**할 때만(그 경우 confidence "언론").

## 3. 출력 스키마 (이 키만, JSON 배열)
```json
{
  "kind": "data",
  "title": "서울 아파트 매매가격지수 +1.06% (2026-05·한국부동산원)",
  "source": "한국부동산원 전국주택가격동향조사",
  "url": "실재 URL",
  "date": "YYYY-MM-DD(기간 말일 또는 발표/결정일)",
  "summary": "수치·기준·전월대비를 한 줄로",
  "category": "price | macro",
  "metric": "지표명(아래 목록에서)",
  "value": 1.06,
  "unit": "% | 건 | 호 | 가구 | 조원 | p",
  "sido": "전국 | 수도권 | 서울 | 경기 | 인천",
  "confidence": "공식 | 언론"
}
```
- **metric 표준명**(이 외 신설 금지): `매매가격지수 변동률`,`전세가격지수 변동률`,`주간 매매변동률`,`주간 전세변동률`,`KB 매매변동률`,`매매전망지수`,`매수우위지수`,`아파트 매매 거래량`,`주택 매매 거래량`,`미분양`,`준공후 미분양`,`입주물량`,`기준금리`,`COFIX`,`주택담보대출 금리`,`가계대출 증감`,`스트레스DSR 가산금리`.
- `value`는 숫자만(부호 포함). 같은 metric·sido는 date가 다르면 시계열로 쌓인다.

## 출력
설명 없이 **JSON 배열만** 한 코드블록으로. 끝에 검증한 출처 / 못 찾은 항목을 짧게 덧붙여라.
적재 명령: `PYTHONPATH=src python3 -m apthub add --file data-out.json --by-date`

=== 프롬프트 끝 ===
