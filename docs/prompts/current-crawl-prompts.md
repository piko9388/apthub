# 현재 사용 중인 챗 크롤 프롬프트 (수동 수집용)

LLM 기반 정성 수집은 자동화하지 않으므로, 아래 프롬프트를 **챗(ChatGPT·Claude 등 웹검색 가능한 모델)** 에
그대로 붙여 넣어 데이터를 모은다. 반환된 JSON 배열을 `data/seed/` 에 새 파일로 저장하면 빌드에 반영된다.

## 공통 규칙 (모든 프롬프트에 적용)
- 출처 URL은 **반드시 실재**(검색으로 확인). 날조 금지, 확인 불가 항목은 제외.
- de-sensationalized: 자극적 표현·단정 제거, 담백한 사실 요약.
- `kind:"data"` 항목은 metric/value/unit/sido/date 정확히. 표준 metric명만 사용.
- 오늘 날짜 기준 미래 시행은 발표일로 date 설정(대시보드가 '예정' 배지 표시).
- 표준 카테고리: `policy | price | macro | semicon` (그 외 금지).
- 표준 지표명(26종): 매매가격지수 변동률·전세가격지수 변동률·주간 매매변동률·주간 전세변동률·
  KB 매매변동률·5분위 평균매매가·5분위 배율·평당가·평형별 실거래가·전세가율·분양가·청약경쟁률·
  경매 낙찰가율·PIR·아파트 매매 거래량·주택 매매 거래량·미분양·준공후 미분양·입주물량·기준금리·
  COFIX·주택담보대출 금리·가계대출 증감·스트레스DSR 가산금리·매수우위지수·매매전망지수
- 표준 밴드: area_band = 40이하/40-60/60-85/85-130/130초과 · price_band = 6억이하/6-9/9-15/15-25/25초과

## A. 공식 지표 시계열 (정량 · price 트랙)
```
너는 한국 수도권(서울·경기·인천) 부동산 공식 지표 시계열 수집 리서처다. WebSearch로 한국부동산원·
국토교통부 통계를 찾아 JSON 배열로 반환한다. 오늘은 {오늘날짜}다.

목표(여러 시점 시계열로):
1. 매매가격지수 변동률(월간, %): 서울·경기·인천·수도권·전국 — 최근 12개월
2. 전세가격지수 변동률(월간, %): 서울·경기·인천·수도권 — 최근 12개월
3. 주간 매매/전세변동률(%): 서울·경기·인천 — 최근 2~3개월 주별
4. 아파트/주택 매매 거래량(건): 서울·경기·인천·전국 — 최근 9개월 월별
5. 미분양·준공후 미분양(호): 전국·수도권·서울 — 최근 9개월 월별
6. 입주물량(가구): 서울·경기·인천·수도권 — 2025/2026 연도 명시

출력 형식(JSON 배열, 항목 예):
{"kind":"data","title":"서울 아파트 매매가격지수 0.8% (2026-03·부동산원)","source":"한국부동산원 전국주택가격동향조사","url":"<실재 URL>","date":"2026-03-31","summary":"...","category":"price","metric":"매매가격지수 변동률","value":0.8,"unit":"%","sido":"서울","confidence":"공식"}
월간은 해당 월 말일, 주간은 기준일. 검증된 것만. 최종 메시지로 JSON 배열만 반환.
```

## B. 금리·대출·시장심리 시계열 (정량 · macro 트랙)
```
너는 부동산 금리·대출·심리 지표 시계열 수집 리서처다. WebSearch로 공식 통계를 찾아 JSON 배열로 반환. 오늘은 {오늘날짜}.

목표(여러 시점 시계열):
1. COFIX(신규취급액, %): 전국 — 최근 9개월 (은행연합회)
2. 주택담보대출 금리(%, 신규취급 가중평균): 전국 — 최근 8개월 (한국은행)
3. 가계대출 증감(조원, 한 기준으로 통일 - 전금융권 또는 은행권): 전국 — 최근 6개월
4. 매수우위지수·매매전망지수(KB, 0~200): 서울·전국 — 최근 6~9개월
5. KB 매매변동률(%): 서울·경기·인천 — 최근 6개월
6. 전세가율(%): 서울·수도권·전국 — 분기별

출력 형식 예:
{"kind":"data","title":"신규 COFIX 2.90% (2026-05·은행연합회)","source":"은행연합회 COFIX","url":"<실재 URL>","date":"2026-05-31","summary":"...","category":"macro","metric":"COFIX","value":2.90,"unit":"%","sido":"전국","confidence":"공식"}
※ 가계대출 증감은 은행권/전금융권을 섞지 말고 한 기준으로만. 검증된 것만. JSON 배열만 반환.
```

## C. 양극화·분양·청약·경매 + 누락 지표 (정량 · price 트랙)
```
너는 수도권 부동산 양극화·분양·청약·경매 지표 수집 리서처다. WebSearch로 공식 통계를 찾아 JSON 배열로 반환. 오늘은 {오늘날짜}.

목표:
1. 5분위 평균매매가(억, KB 1·5분위)·5분위 배율(배): 서울·전국 — 분기/월별
2. 평당가(만원/3.3㎡): 서울·경기·인천 — 분기별 (KB/부동산원)
3. 평형별 실거래가(억, 면적대별): 서울 — area_band 필드 포함
4. PIR(배): 서울·경기·인천·전국 — 분기/연도
5. 분양가(만원/평): 서울·수도권·전국 — 최근 6개월 월별 (HUG)
6. 청약경쟁률(대1): 서울·수도권 — 월별 (청약홈)
7. 경매 낙찰가율(%): 서울·경기·인천·전국 — 최근 6개월 월별 (지지옥션)

출력 형식 예:
{"kind":"data","title":"서울 아파트 5분위 평균 25.1억 (2026-06·KB)","source":"KB부동산 월간 주택가격동향","url":"<실재 URL>","date":"2026-06-30","summary":"...","category":"price","metric":"5분위 평균매매가","value":25.1,"unit":"억","sido":"서울","price_band":"25초과","confidence":"공식"}
㎡당→평당 환산값은 confidence "추정"으로 표기. 검증된 것만. JSON 배열만 반환.
```

## D. 정책·거시·반도체 뉴스 (정성 · 공식/주요언론 위주)
```
너는 수도권 부동산 정책·거시·반도체 뉴스 리서처다. 지금 즉시 너 스스로 WebSearch를 15회 이상
직접 실행해 기사를 수집한다(위임·대기 금지). 시세/실거래 기사는 제외. 오늘은 {오늘날짜}.

분야:
1. policy 25~35건: 가계부채 대책 후속·스트레스 DSR·토지거래허가·세제(양도/취득/종부세)·
   정비사업·공급(3기 신도시·수도권)·임대차·청약제도·규제지역 지정/해제
2. macro 15~25건: 금통위·기준금리 전망·가계부채 거시영향·환율/물가의 부동산 영향·대출 총량규제
3. semicon 10~15건: SK하이닉스·삼성 실적/투자/HBM·용인/평택/청주/이천 클러스터·성과급→배후수요

출처 우선순위: A급(국토부·금융위·한은·기재부·부동산원 보도자료) > B급(연합·한경·매경·서경·머투·
뉴시스·뉴스1·뉴스핌·비즈한국). 지양: zippoo·hogangnono·apt2·aptbong·namu.wiki 등 집계/커뮤니티/위키.

출력 형식 예:
{"title":"금융위, 스트레스 DSR 3단계 전 금융권 확대","source":"금융위원회","url":"<실재 URL>","date":"2026-06-15","summary":"담백한 사실 요약","category":"policy","comment":"시장 함의 한 줄"}
출처 URL은 검색으로 확인한 실재 URL만. 날조 금지. 50건 이상. JSON 배열만 반환.
```

## 적재 방법
1. 챗이 반환한 JSON 배열을 `data/seed/` 에 새 파일로 저장 (예: `data/seed/news-2026-07.json`).
2. 중복/충돌 점검: 같은 `metric+sido+date` 가 기존에 있으면 제외, 하위지역 변동률(동탄·분당 등)은
   시도 시계열을 왜곡하므로 `kind:"data"` 대신 일반 뉴스로 적재.
3. 빌드: `APTHUB_PUBLIC=1 PYTHONPATH=src python3 scripts/build_site.py`
4. 커밋·푸시 → `pages.yml` 자동 배포.
