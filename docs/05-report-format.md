# 05 — 리포트 포맷 (데일리 / 위클리)

> 톤: 간결·실행지향. 각 항목은 1~2줄. 모든 시그널에 "내 매수계획 함의" 한 줄 필수.

## 데일리 (평일 아침)
```
📅 [YYYY-MM-DD] 정훈 부동산 시그널 — Daily

🔴 핵심 트리거 (있을 때만)
- [내용] → 함의: [내 천장/타이밍에 미치는 영향 1줄]

📊 강서 시세 스냅
- 등촌주공3·5 / 가양 / 발산 / 염창 / 대방: 신규 실거래·호가 변동만
- 주담대 금리·기준금리: 변동 시만

📰 정책/뉴스 Top 3
1. [제목] (출처) — 1줄 요약 → 함의 1줄
2. …
3. …

💡 오늘의 액션 (있을 때만)
- 예: 은행 가심사 문의 / 관심단지 매물 확인
```

## 위클리 (일요일)
```
🗓️ [YYYY 'W##' 또는 M월 #주차] 정훈 부동산 시그널 — Weekly

1) 정책 변화 & 내 천장 영향
   - 이번 주 대출·세제·규제 변화 → 8월 8.5억 / 27.2 10.5억에의 영향

2) 강서 관심단지 주간 시세 표
   | 단지 | 전용59 | 주간변동 | 메모 |
   | 등촌주공3 / 등촌주공5 / 가양 / 발산 / 염창 / 대방 |

3) 금리·거시·반도체
   - 기준금리·코픽스·전세시장 / SK하이닉스·HBM·메모리(성과급 재원)

4) 부읽남 관점 코멘트
   - 이번 주 시장을 부읽남 38강 원칙으로 1문단 해석
     (대안의 부재 / 계획-과정-현실화 / 능력의 객관화 / 시간 레버리지 / 안전판)

5) 다음 주 체크포인트 / 액션
   - 발표 예정 정책·지표, 내가 할 일
```

## 발송
- 채널: 개인 메일 또는 텔레그램 봇.
- 형식: Markdown(메일은 HTML 변환). 길이 데일리 ½페이지·위클리 1페이지 이내.
- 트리거 없는 날은 "특이사항 없음 + 시세 스냅"만 짧게.
