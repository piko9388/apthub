# 02 — 관심 지역·단지

> 크롤러가 시세·뉴스를 필터링할 핵심 지명/단지. **강서구 우선**, 인접 동작·마곡 보조.

## 1순위 — 서울 강서구
| 단지 | 전용59 시세(추정) | 관심 사유·호재 |
|---|---|---|
| **등촌주공3단지** | ~11.6억 | 1994년 구축, 정비(재건축)·고도제한 완화, 9호선 급행, CJ부지(스타필드) 호재. *현 자금 천장 초과 → 보류·관찰* |
| **등촌주공5단지** | ~11.0억 | 등촌주공3과 동일 호재권. *천장 초과 → 관찰* |
| **가양** 대단지 | 9~11억 | CJ제일제당 부지 개발(스타필드)·9호선. 대단지 |
| **발산** | 9~10억 | 5호선·마곡 인접, 정훈 통근 양호 |
| **염창** | ~10억 | 9호선 염창역, 여의도 최단 |
| **우장산** | 9~10억 | 5호선, 학군·주거 안정 |

## 2순위 — 인접권
| 지역 | 사유 |
|---|---|
| 동작 **대방** | 5호선 여의도(희주)+올림픽대로(정훈)+보라매공원. 통근 균형 |
| **보라매** | 대방 인접 |
| **마곡** | 업무지구·신축이나 가격대 상방 |

## 보유/제외
- 보유: **검단 e편한세상 웰카운티**(인천 서구 원당동) — 부모 거주, 매도 대상 아님. 검단 시세·정책은 참고만.
- **제외**: 경기 비규제(동탄 등) — 희주 통근·부모 육아 지원 불가.
